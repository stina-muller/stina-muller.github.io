What is grid based design?
A grid based design uses grids to help form the layout of your webpage. Understanding how a grid works will enable you to be able to know how to layout your webpage.

Whats all the hype about responsive webpages?
Due to the growing number of devices we all use, actually determines how we interact with a webpage, and how that webpage responds to the all these different screen sizes. If your webpage is not responsive then the users experience of your webpage can be unpleasant and/or confusing, which can be off putting when the user is trying to interact with your page.

What is semantic structure?
This is the way that you name your tags that you use in your code that describe what you are naming and why, so that it is clear to understand what that tag is just by looking at it.
